This file is pure host-side inspection. It tells you what machine your kernels will run on.

What It Teaches

  • How to call the CUDA Runtime API from a tiny program.
  • What a device count and device properties object look like.
  • Why hardware facts matter before we talk about tiles, warps, and performance.

What To Read In The Code

  1. CHECK_CUDA(cudaGetDeviceCount(...))
  2. cudaGetDeviceProperties(...)
  3. How host-side printing turns raw device properties into human-readable facts.
This file is not about optimization. It is about grounding.
