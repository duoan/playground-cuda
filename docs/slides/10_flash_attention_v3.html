10 FlashAttention v3

Pipeline Thinking Gets Deeper

v3 pushes the same attention math into a more explicit pipeline. The teaching goal is to make modern staged kernels feel readable, not mysterious.

Source: src/cuda/10_flash_attention_v3.cu

Ping-Pong Tile Pipeline

Two-stage mental model
tile 0
load s0
compute s0
write O
tile 1
wait
load s1
compute s1
write O
tile 2
wait
load s0
compute s0
The important lesson is not the exact number of stages. It is the overlap: while one stage computes, another stage prepares the next tile.
Code that matches the pipeline
int load_stage = tile_index % 2;
int compute_stage = 1 - load_stage;

preload(next_tile, load_stage);
__syncthreads();
consume(current_tile, compute_stage);
This is the teaching bridge toward real async copy / tensor-core pipelines.

How to read v3

More staging Data movement becomes a first-class design concern.
