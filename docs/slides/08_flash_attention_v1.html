The score tile exists only briefly inside the loop. After it updates softmax stats and output, the kernel moves on.
What must be carried across tiles
m Running row max for numerical stability.
l Running denominator of softmax.
O Running output vector, rescaled when the max changes.
float new_m = fmaxf(old_m, tile_max);
float scale_old = expf(old_m - new_m);
float scale_new = expf(tile_max - new_m);
new_l = old_l * scale_old + tile_l * scale_new;
These three state variables are the heart of online softmax.
