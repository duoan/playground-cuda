05 LayerNorm

Row Statistics Plus Affine Transform

LayerNorm is a perfect lesson in reduction-based deep-learning kernels: compute mean, compute variance, normalize, then apply gamma and beta.

Source: src/cuda/05_layernorm.cu

float mean = shared_sum[0] / cols;
float inv_std = 1.0f / sqrtf(shared_var[0] / cols + eps);

for (int col = threadIdx.x; col < cols; col += blockDim.x) {
  float normalized = (row_ptr[col] - mean) * inv_std;
  out_ptr[col] = normalized * gamma[col] + beta[col];
}
LayerNorm is “two reductions plus affine transform,” all done row by row.

Kernel Skeleton

  • Reduction 1: row mean
  • Reduction 2: row variance
  • Elementwise step: normalize and apply gamma/beta
If softmax was “two reductions plus normalize,” layernorm is “two reductions plus affine normalize.”

Why It Matters

Transformers use normalization constantly. If you understand row-wise layernorm well, later fused kernels will feel much less magical.
